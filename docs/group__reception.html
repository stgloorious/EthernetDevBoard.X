◆ enc424j600_getPacketCount()

uint8_t enc424j600_getPacketCount ( )

Reads out internal register PKCNT;.

Needs to be polled to check for new packets. If it is greater zero, there are unprocessed packets. To decrement the packet count (after processing) SETPKTDEC needs to be set (via single byte command).

Returns
Value of PKCNT
