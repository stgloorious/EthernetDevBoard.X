◆ ARP_LINK_BROADCAST

#define ARP_LINK_BROADCAST   true

Replies are sent using a link-level broadcast and not unicast.

Enabling this option leads to a quicker conflict detection, but does also increase traffic.

Warning
NOT RECOMMENDED for general use, but other specification may choose to use broadcast for ARP replies if appropriate.
See also
RFC 5227, Section 2.6, Page 14f.
