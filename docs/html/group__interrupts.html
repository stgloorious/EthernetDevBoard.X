◆ ENC424J600_getInterruptFlags()

uint16_t ENC424J600_getInterruptFlags ( )

Reads out the Interrupt Flag Register EIR.

Note that these flags are always valid, no matter if the interrupt is enabled or not. This way the interrupt flags can be polled periodically for low priority stuff. Note that the MSB is not an interrupt flag (it is returned, too).

Returns
The value of the Interrupt FLag Register EIR
