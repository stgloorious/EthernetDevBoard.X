EthernetDevBoard  v1.0
The goal of the project "Ethernet Development Board" is to get a grasp on how a network operates. It uses a 8-Bit Microcontroller PIC18F25K50 and an Ethernet Controller ENC424J600 to achieve 100Base-TX Ethernet connectivity.
