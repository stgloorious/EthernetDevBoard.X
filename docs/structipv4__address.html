Detailed Description

IPv4 Address type.

Address is stored in an array of bytes, the first element being the first byte if read from left to right.

// This sets an IPv4 Address to 192.168.0.1
ipv4_address_t ip;
ip[0] = 192;
ip[1] = 168;
ip[2] = 0;
ip[3] = 1;
